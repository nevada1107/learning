夏休みを犠牲に


みんなが海や川で遊んだり、バーベキューをしたりして夏を満喫している中、一人部屋で、夏休みの1/3ほどを犠牲にしてパソコン とにらめっこをしている男がここに一名。そもそも外で遊ぶのは、、、、嫌いではないですけど、家を1歩出た瞬間どうでもよくなるんです。 夏休みは9/26までですけどもこれまでにどれだけ進歩できるでしょうか。。。。



第1の壁　そもそも要素を知らない



htmlといえば要素。要素といえばhtmlと言ってもいいような切っても切れない関係の要素。要素ってなんなん？って方、HTMLをかくには

< html >
みたいな感じな要素と呼ばれるもので文書をくくってくくって何度も繰り返してそれをCSSと呼ばれるもので さらにスタイリングをして完成です。基礎的な要素を覚えるのに本当に2か月以上かかりました。ノートにまとめては実行。。を何度も繰り返し。

第2の壁　iframeむずすぎ！そしてdiv有能やな！



iframeというのはhtmlの中にさらにhtmlを埋め込めることができる汎用的要素。駆使できるようになるとまた可能性が広がるようです。 でもテーブルっていう要素のほうが簡単でいいです。で、汎用性っていったら、divとspanとclassが有能すぎ！。。最近の身近なできごとで例えたら 1000人の人がいる中で、60代以上を"高齢者"という名前のグループでくくって、"高齢者"にコロナワクチンを打つ...みたいな感じに"くくる" という行為が可能になるわけです。


第3の壁　cssとJavascript、こんにちは



cssっていうのはみんな大好きマインクラフトで例えるところのテクスチャ―パック的な意味合いです。すきなようにくみたててそれをさらに スタイリングする感じです。そしてJavaScript、これはもうModです。動作を加えることが可能です。日付や時間を取得したり、計算も可能です。 これがもう、cssはまだしも、Javascriptが本当に奥が深くて、汎用性が非常に高く、楽しいものの難しい。そんなものです。
